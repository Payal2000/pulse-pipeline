## About

AI that doesn't just provide answers — it helps you take action.

Build an agent that can reason, plan, and execute tasks under your oversight. You provide the logic; Gemini provides the brain; partners provide the superpowers.

## Requirements

### The Challenge

Engineer a functional agent that solves a real-world challenge — specifically targeting work, personal life, hobbies, or daily routines.

### Goals

- **Move Beyond Chat:** The agent shouldn't just answer questions. It should use tools and capabilities to accomplish tasks (e.g., managing a local database, automating a hobbyist workflow, or interacting with a live web service).
- **The Multi-Step Mission:** Build a system that can handle complex goals. The agent should plan the steps and use the tools at its disposal to finish the job, while keeping the user in control.
- **Partner Power:** The solution must demonstrate a meaningful integration with at least one participating partner's solution using MCP.

### How to Build

- **Google Cloud Agent Builder:** Use this low-code, high-power environment to build and orchestrate agents with grounding in your own data.

## What to Build

Build a functional agent — powered by Gemini and Google Cloud Agent Builder — that integrates a Partner Entity's MCP server to solve a real challenge.
